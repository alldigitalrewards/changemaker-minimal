# Troubleshooting Guide

Find solutions to common issues and errors you may encounter while using Changemaker.

## Quick Diagnostic Checklist

Before diving into specific issues, try these general troubleshooting steps:

- [ ] Refresh the page (Ctrl+R or Cmd+R)
- [ ] Clear browser cache and cookies
- [ ] Try a different browser (Chrome recommended)
- [ ] Check if you're logged in with correct account
- [ ] Verify you have appropriate role/permissions
- [ ] Check workspace URL is correct
- [ ] Review [status page](https://status.changemaker.im) for outages

## Authentication Issues

### Cannot Log In

**Symptoms:**
- Login fails with "Invalid credentials"
- Stuck on login page
- Redirected back to login after entering credentials

**Common Causes and Solutions:**

**Incorrect Email or Password**
1. Double-check email address (no typos)
2. Ensure caps lock is off
3. Try **Forgot Password** to reset
4. Check if you have multiple accounts

**Account Not Verified**
1. Check email for verification link
2. Look in spam/junk folder
3. Request new verification email
4. Contact support if no email received

**Browser Issues**
1. Clear browser cache: Settings → Privacy → Clear browsing data
2. Disable browser extensions temporarily
3. Try incognito/private mode
4. Try different browser

**Session Expired**
1. Simply log in again
2. Check "Remember me" if desired
3. This is normal after 30 days inactive

### Logged Out Unexpectedly

**Causes:**
- Session expired (timeout after 24 hours inactive)
- Logged in from another device
- Browser cleared cookies
- Security logout due to suspicious activity

**Solutions:**
1. Log back in (your data is safe)
2. Enable "Remember me" option
3. Use password manager for quick re-login
4. Contact support if happens frequently

### "Access Denied" Errors

**Symptoms:**
- 403 Forbidden error
- "You don't have permission to access this"
- Redirected to access denied page

**Solutions:**

**Check Your Role**
1. Go to **Profile → Workspace**
2. Verify your role (Admin vs. Participant)
3. Some pages require Admin access
4. Request role upgrade from workspace admin

**Verify Workspace Access**
1. Confirm you're member of the workspace
2. Check invitation was accepted
3. Verify workspace URL is correct
3. Contact workspace admin to verify membership

**Check Resource Permissions**
1. Challenge may be private/invite-only
2. Document may have role restrictions
3. Feature may be admin-only
4. Ask admin to grant access

## Challenge Issues

### Cannot See Challenges

**No Challenges Listed**

**Causes and Solutions:**

1. **No active challenges exist**
   - Wait for admin to create challenges
   - Check back later
   - Ask admin when challenges will be available

2. **You're not in the workspace**
   - Verify workspace membership
   - Accept pending invitation
   - Contact admin for access

3. **Challenges are invite-only**
   - Look for invitation email
   - Ask admin to send invite
   - Check if you meet eligibility criteria

4. **Wrong workspace**
   - Verify you're in correct workspace
   - Check workspace URL
   - Switch workspace if needed

### Cannot Enroll in Challenge

**Enrollment Button Disabled or Missing**

**Reasons:**

1. **Already enrolled**
   - Check your enrollments
   - View challenge from participant dashboard
   - Cannot enroll twice in same challenge

2. **Challenge hasn't started**
   - Check start date
   - Cannot enroll before start date
   - Sign up for notifications

3. **Challenge has ended**
   - Past end date
   - Enrollments closed
   - Watch for future similar challenges

4. **Invite-only challenge**
   - Need specific invitation link
   - Ask admin for access
   - Check eligibility requirements

5. **Already at capacity**
   - Challenge has enrollment limit
   - Limit already reached
   - Join waitlist if available

### Challenge Not Loading

**Page Shows Error or Blank**

**Solutions:**

1. **Check URL**
   - Verify challenge ID is correct
   - No typos in URL
   - Try copying link again

2. **Challenge was deleted**
   - Admin may have removed it
   - Check with workspace admin
   - Review other available challenges

3. **Browser issues**
   - Refresh page
   - Clear cache
   - Try different browser

4. **Network problems**
   - Check internet connection
   - Try different network
   - Disable VPN temporarily

## Reward Issues

### Reward Email Not Received

**Didn't receive reward notification**

**Check These First:**

1. **Spam/Junk Folder**
   - Check spam folder thoroughly
   - Add noreply@rewardstack.com to contacts
   - Mark as "Not Spam"

2. **Incorrect Email Address**
   - Verify email in your profile
   - Update if incorrect
   - Ask admin to resend to correct email

3. **Email Delay**
   - Can take up to 30 minutes
   - Check again later
   - Contact support if >2 hours

4. **Email Server Blocking**
   - Corporate email may block
   - Try personal email
   - Ask IT to whitelist RewardStack

**Request Resend:**
1. Contact workspace admin
2. Admin can resend from reward history
3. Or issue replacement reward

### Cannot Claim Reward

**Claim Link Not Working**

**Common Issues:**

1. **Link Expired**
   - Some rewards expire after 90 days
   - Request replacement from admin
   - Claim future rewards promptly

2. **Browser Problems**
   - Clear cache and cookies
   - Try different browser
   - Disable extensions
   - Use incognito mode

3. **Geographic Restriction**
   - Reward not available in your country
   - Request different reward type
   - Admin can issue alternative

4. **Already Claimed**
   - Can only claim once
   - Check if you claimed previously
   - Contact support if you didn't

**Get Help:**
- Email workspace admin
- Include transaction ID from email
- Describe exact error message
- Contact RewardStack support

### Wrong Reward Amount

**Received Different Value Than Expected**

**Verify First:**

1. Check challenge reward settings
2. Review completion criteria
3. Confirm you met all requirements
4. May be tiered based on performance

**If Actually Wrong:**
1. Screenshot the discrepancy
2. Note expected vs. received amount
3. Contact workspace admin with details
4. Admin can issue additional reward if error confirmed

## Performance Issues

### Pages Loading Slowly

**Symptoms:**
- Long load times
- Spinning indicators
- Timeouts

**Solutions:**

**Browser Optimization**
1. Close unnecessary tabs
2. Clear browser cache
3. Disable heavy extensions
4. Update browser to latest version
5. Restart browser

**Network Issues**
1. Check internet speed
2. Try different network
3. Disable VPN if using
4. Move closer to WiFi router
5. Use wired connection if possible

**Device Performance**
1. Close other applications
2. Restart computer
3. Free up disk space
4. Check system resources

**Changemaker Side**
1. Check [status page](https://status.changemaker.im)
2. May be temporary high traffic
3. Try again in a few minutes
4. Report if persists

### Images Not Loading

**Missing Images or Broken Icons**

**Quick Fixes:**

1. **Refresh page** (Ctrl+R or Cmd+R)
2. **Clear cache**
3. **Check network connection**
4. **Disable ad blockers** temporarily
5. **Try different browser**

**Persistent Issues:**
- Report to support with:
  - Page URL
  - Browser and version
  - Screenshot of issue

### Dashboard Not Updating

**Seeing Stale Data**

**Causes:**

1. **Browser Cache**
   - Force refresh: Ctrl+Shift+R (Cmd+Shift+R on Mac)
   - Clear cache completely
   - Try incognito mode

2. **Real-time Sync Delay**
   - Can take 30-60 seconds
   - Wait a moment then refresh
   - Check again in a few minutes

3. **JavaScript Error**
   - Open browser console (F12)
   - Look for errors in red
   - Screenshot and report to support

## Data Issues

### Missing Enrollments or Submissions

**Data Appears to Be Lost**

**Before Panicking:**

1. **Verify Workspace**
   - Confirm correct workspace
   - Check workspace switcher
   - May be in different workspace

2. **Check Filters**
   - Remove any active filters
   - Select "All" for status
   - Clear date range filters

3. **Verify Permissions**
   - Some data may be role-restricted
   - Admin sees all, participants see own
   - Request admin to verify

**Actually Missing:**
1. Don't panic - we have backups
2. Contact support immediately
3. Provide:
   - What's missing
   - When you last saw it
   - Any error messages
   - Steps that led to issue

### Incorrect Data Displaying

**Wrong Numbers, Dates, or Information**

**Check:**

1. **Time Zone Settings**
   - Dates shown in your local time zone
   - Verify in profile settings
   - Can change if incorrect

2. **Cached Data**
   - Force refresh page
   - Clear browser cache
   - View in incognito mode

3. **Recent Changes**
   - Admin may have updated
   - Data sync in progress
   - Wait a moment and check again

**Report If Persists:**
- Screenshot the incorrect data
- Note what it should be
- Include affected URLs
- Contact support

## Integration Issues

### RewardStack Connection Failed

**Cannot Connect RewardStack**

**Admin Solutions:**

1. **Verify API Credentials**
   - Check Client ID is correct
   - Verify Client Secret (no typos)
   - Ensure no extra spaces
   - Credentials not expired

2. **Check Environment**
   - Match sandbox vs. production
   - Credentials must match environment
   - Test in sandbox first

3. **RewardStack Account Status**
   - Account fully verified
   - Not suspended or limited
   - Balance sufficient
   - Check RewardStack dashboard

4. **Reconnect Integration**
   - Disconnect current integration
   - Wait 30 seconds
   - Reconnect with fresh credentials
   - Test connection

### Webhook Not Working

**Not Receiving Webhook Events**

**Verify:**

1. **Webhook URL is Correct**
   - Check for typos
   - Must be publicly accessible
   - Must accept POST requests
   - Must respond with 200 OK

2. **Webhook is Active**
   - Check webhook status
   - Ensure not disabled
   - Verify events selected
   - Test with webhook tester

3. **Network Configuration**
   - Firewall not blocking
   - SSL certificate valid
   - DNS configured correctly
   - Server running and accessible

**Debug:**
1. Check webhook logs
2. Test with curl or Postman
3. Verify payload format
4. Check error responses

## Mobile Issues

### Mobile Layout Problems

**App Doesn't Look Right on Phone**

**Quick Fixes:**

1. **Rotate Device**
   - Try portrait and landscape
   - Some features work better in landscape

2. **Update Browser**
   - Ensure latest version
   - Mobile Chrome recommended
   - Safari on iOS works well

3. **Clear Mobile Cache**
   - Settings → Browser → Clear data
   - Restart browser
   - Try again

4. **Desktop Mode**
   - Some features work better in desktop mode
   - Enable in browser settings
   - Temporary workaround

### Touch/Tap Not Working

**Buttons Don't Respond to Tap**

**Try:**

1. **Tap More Deliberately**
   - Firm, quick tap
   - Not too hard or too soft
   - Dead center of button

2. **Scroll First**
   - Page may still be loading
   - Wait for load complete
   - Then tap

3. **Remove Screen Protector**
   - May interfere with touch
   - Try without temporarily
   - Clean screen

4. **Try Different Finger**
   - Dry finger works better
   - Remove gloves
   - Clean screen

## Error Messages

### "Something Went Wrong"

**Generic Error Message**

**Immediate Steps:**

1. Refresh page
2. Try action again
3. Clear cache
4. Try different browser

**If Persists:**
1. Open browser console (F12)
2. Screenshot any errors in console
3. Note exact steps that caused error
4. Report to support with:
   - Screenshot
   - Browser and version
   - Steps to reproduce
   - What you were trying to do

### "Network Error"

**Connection Failed Messages**

**Diagnose:**

1. **Check Internet Connection**
   - Visit other websites
   - Run speed test
   - Restart router
   - Try different network

2. **VPN/Proxy**
   - Disable VPN temporarily
   - Disable proxy settings
   - Try direct connection
   - Whitelist Changemaker

3. **Firewall/Security**
   - Corporate firewall may block
   - Antivirus may interfere
   - Add exception for changemaker.im
   - Contact IT if needed

### "Invalid Request" or 400 Error

**Bad Request Errors**

**Common Causes:**

1. **Invalid Input**
   - Check form fields
   - No special characters where not allowed
   - Required fields filled
   - Correct format (email, date, etc.)

2. **Expired Session**
   - Log out and log back in
   - Session may have expired
   - Try action again

3. **Browser Cache**
   - Clear cache
   - Hard refresh
   - Try incognito

## API Issues

### API Rate Limit Exceeded

**Error:** "Too many requests"

**Solutions:**

1. **Wait and Retry**
   - Rate limit resets every minute
   - Wait 60 seconds
   - Try request again

2. **Reduce Request Frequency**
   - Batch requests where possible
   - Add delays between requests
   - Cache responses

3. **Contact Support**
   - Need higher rate limit
   - Discuss your use case
   - May upgrade limit

### Invalid API Key

**Error:** "Unauthorized" or 401

**Fix:**

1. **Check API Key**
   - Copy key carefully (no extra spaces)
   - Not expired
   - Correct key for environment

2. **Regenerate Key**
   - Go to Settings → API
   - Generate new key
   - Update in your application
   - Old key stops working immediately

3. **Verify Permissions**
   - API key has required scopes
   - Account has necessary role
   - Workspace access is active

## Browser-Specific Issues

### Issues in Safari

**Known Safari Quirks:**

1. **Third-Party Cookies**
   - Safari blocks by default
   - Preferences → Privacy → Disable "Prevent cross-site tracking"
   - Required for some features

2. **Intelligent Tracking Prevention**
   - Can interfere with login
   - Add changemaker.im to exceptions
   - Or use Chrome for better experience

### Issues in Firefox

**Firefox-Specific:**

1. **Enhanced Tracking Protection**
   - May block some features
   - Click shield icon in address bar
   - Disable for changemaker.im
   - Or switch to Standard protection

### Issues in Edge

**Edge-Specific:**

1. **Compatibility Mode**
   - Ensure not in IE mode
   - Use modern Edge (Chromium-based)
   - Update to latest version

## Getting Additional Help

### Before Contacting Support

Gather this information:

1. **Account Details**
   - Your email address
   - Workspace name/slug
   - Your role

2. **Issue Description**
   - What happened
   - What you expected to happen
   - When it started

3. **Environment**
   - Browser and version
   - Operating system
   - Device type
   - Internet connection type

4. **Evidence**
   - Screenshots
   - Error messages (exact text)
   - Console errors (if technical)
   - Steps to reproduce

### Contact Support

**Email Support**
- Address: [support@changemaker.im](mailto:support@changemaker.im)
- Include all gathered information
- Attach screenshots
- Response within 24 hours

**Emergency Support**
- For critical production issues
- Workspace completely down
- Data loss
- Security concerns
- Email with "URGENT" in subject

**In-App Chat**
- Click chat icon (bottom right)
- Available during business hours
- Quick questions and guidance
- Escalated to email if needed
