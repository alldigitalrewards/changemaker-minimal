# Participant Management Guide

Everything you need to know about managing participants in your Changemaker workspace.

## Overview

Participants are the heart of your innovation programs. This guide covers how to:
- Invite and onboard participants
- Manage participant access
- Track participation and engagement
- Handle participant data and privacy

## Understanding Participant Roles

### Participant Role

Standard participants can:
- Browse available challenges
- Enroll in challenges
- Submit work
- View their own progress
- Claim rewards
- Access participant documentation

### Admin Role

Workspace admins can:
- All participant capabilities
- Create and manage challenges
- View all participants
- Issue rewards
- Configure workspace settings
- Access admin documentation

## Inviting Participants

### Email Invitations

The most common way to add participants:

1. Navigate to **Settings → Team**
2. Click **Invite Member**
3. Enter participant email address
4. Select role: **Participant** or **Admin**
5. Click **Send Invitation**

The participant receives an email with:
- Invitation link
- Workspace name
- Role assignment
- Getting started information

### Bulk Invitations

For inviting multiple participants:

1. Prepare CSV file with columns:
   ```csv
   email,role,firstName,lastName
   john@example.com,PARTICIPANT,John,Doe
   jane@example.com,PARTICIPANT,Jane,Smith
   ```
2. Go to **Settings → Team → Bulk Import**
3. Upload CSV file
4. Review import preview
5. Confirm and send invitations

### Public Enrollment Links

For open challenges:

1. Navigate to challenge settings
2. Enable **Public Enrollment**
3. Copy enrollment link
4. Share via:
   - Email campaigns
   - Slack/Teams channels
   - Internal portals
   - Social media

Anyone with the link can:
- Create an account
- Join the workspace
- Enroll in the challenge

## Onboarding Participants

### First-Time Setup

When participants first join:

1. **Account Creation**
   - Sign up with email
   - Set password
   - Verify email address
   - Complete profile

2. **Workspace Orientation**
   - View workspace overview
   - See available challenges
   - Access documentation
   - Review guidelines

3. **Challenge Discovery**
   - Browse active challenges
   - Read descriptions
   - Check requirements
   - Enroll in challenges

### Onboarding Best Practices

✅ **Recommended:**
- Send welcome email with clear next steps
- Provide workspace overview document
- Host kickoff meetings for major programs
- Create onboarding challenge for practice
- Assign buddy or mentor
- Share FAQ and support contacts

❌ **Avoid:**
- Overwhelming with too much information
- Skipping the welcome message
- Forgetting to explain how to get help
- Not setting clear expectations

## Managing Participant Access

### Viewing All Participants

Access participant list:

1. Go to **Admin → Participants**
2. View table with:
   - Name and email
   - Role
   - Join date
   - Active enrollments
   - Completion rate
   - Last activity

### Filtering and Search

Find specific participants:
- **Search**: By name or email
- **Filter by role**: Admin, Participant
- **Filter by status**: Active, Inactive
- **Sort**: By join date, activity, completions

### Updating Participant Information

For each participant:
1. Click participant name
2. View detailed profile
3. Click **Edit**
4. Update information:
   - Name
   - Email
   - Role
   - Status
5. Save changes

### Removing Participants

When someone leaves:

1. Navigate to participant profile
2. Click **Remove from Workspace**
3. Choose what happens to their data:
   - **Archive**: Keep submissions, hide profile
   - **Delete**: Remove all data (GDPR compliance)
4. Confirm action

**Note**: This cannot be undone. Consider archiving instead of deleting.

## Communication with Participants

### Individual Messages

Send direct messages:
1. Navigate to participant profile
2. Click **Send Message**
3. Compose message
4. Send or schedule

### Broadcast Announcements

Send to all participants:
1. Go to **Admin → Communications**
2. Click **New Announcement**

4. Select audience:
   - All participants

5. Send or schedule

## Data Privacy and Security

### GDPR Compliance

Changemaker helps you comply with GDPR:

**Data Access**
- Participants can download their data
- Available in **Profile → Privacy**
- Includes all submissions and rewards

**Data Deletion**
- Right to be forgotten
- Complete data removal option
- Irreversible process

**Data Portability**
- Export in standard formats (JSON, CSV)
- Includes all personal data
- Easy to transfer

### Data Security

**Access Controls**

- Workspace isolation

**Data Encryption**
- SSL/TLS for data in transit

- Secure database storage

### Managing Consent

**Consent Tracking**
- Terms of service acceptance
- Privacy policy acceptance
- Marketing communications opt-in

**Updating Consent**

## Troubleshooting

### Participant Can't Access Workspace

**Check:**
1. Email verification completed
2. Invitation not expired
3. Correct workspace URL
4. Account status is active
5. No technical issues

**Solutions:**
- Resend invitation
- Verify email address
- Check spam folder
- Update account status
- Contact support

### Low Participation Rates

**Diagnose:**
- Are challenges appealing?
- Are requirements too complex?
- Is communication effective?
- Are rewards competitive?

**Improve:**
- Survey participants for feedback
- Simplify challenge requirements
- Increase communication frequency
- Review reward structure
- Make enrollment easier

### Participant Data Issues

**Common problems:**
- Duplicate accounts
- Incorrect role assignments
- Missing profile information
- Enrollment errors

**Solutions:**
1. Merge duplicate accounts via support
2. Update roles in participant settings
3. Request profile completion
4. Check enrollment permissions
